HTML Signatures in Apple Mail

Short, concise and good looking HTML email signatures!

Though there are quite a few of email applications for the Mac OSX none of them combine all the email accounts into one. I mean they don’t work with my work email stringhamdb@familysearch.org while working with my personal and schools emails or vice versa. The only one that does is Apple Mail. The email signature feature in Apple Mail is very limited and this post is a way to work around that.

A very nicely detailed post “How To Make an HTML Signature in Apple Mail for Yosemite OS X 10.10” explains all you need to know. I’ll just explain the steps I took:

  • I then minified the HTML at: http://www.willpeavy.com/minifier/

  • Next I created a new email signature in Apple Mail (Mail > Preferences > Choose Email Account > +)

  • In that new email signature it put the text “PLACEHOLDER”.

  • Exited Apple Mail.

  • Navigated to: ~/Library/Mail/V2/MailData/Signatures

  • Selected the desired email signature file. It should look something like 04868CEC-66BF-461B-A566-B74574C880EB.mailsignature.

  • Right click the desired email signature file and make sure the checkbox Locked is NOT checked.

  • Open the desired email signature file in a text editor. I use Sublime Text 3.

  • Paste the minified HTML into the email signature file on top of the text “PLACEHOLDER”.

  • Save the email signature file.

  • On the file information dialog make sure Locked is now checked.

  • Launch Apple Mail, compose an email and you should see your nifty new HTML email signature.
